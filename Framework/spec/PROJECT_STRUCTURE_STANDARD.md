# Project Structure Standard

Formal project:

```text
<Project>/
├── README.md
├── Current_Production/
├── Reports/
├── Audit_Summary/
├── spec/
├── prompts/
├── scripts/
└── Archive/
```

Library project:

```text
<Project>/
└── Current_Production/
    ├── Library/
    └── Host_Sheet/
```

Standalone script:

```text
General/Scripts/
```
