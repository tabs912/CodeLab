# Codex Startup Procedure

1. Read the repository `README.md`.
2. Read the repository `AGENTS.md`.
3. Identify the active project or script.
4. Determine the review level.
5. Read applicable Framework standards.
6. Read project-specific documentation when it exists.
7. Identify the governing production source.
8. Review current reports when requested.
9. Ignore excluded folders.
10. Do not modify code unless implementation is requested.

If a requested file is not visible:

```bash
./tools/sync_workspace.sh
```

Then search again before reporting it missing.
