# CodeLab Codex Startup Procedure

1. Read `README.md`.
2. Read `AGENTS.md`.
3. Identify the active project or script.
4. Determine Quick, Standard, or Exhaustive Review.
5. Review applicable root `spec/` documents.
6. Review project-specific documentation when it exists.
7. Identify `Current_Production/` when applicable.
8. Review current project reports when requested.
9. Ignore `Archive_To_Move/`.
10. Do not modify code unless implementation is requested.

If a requested artifact cannot be found:

```bash
./tools/sync_workspace.sh
```

Then search again before reporting it missing.
